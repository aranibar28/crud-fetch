## Getting started

### Pre-requisitos 📋
Verificar la versión del nodeJS y npm
```
node -v
npm -v
```

### Instalación 🔧
Instalar los paquetes `node_modules`
```
npm install
```

Ejecutar servidor falso
```
npm run server
```

### Comenzar 🚀
- Dirigirse http://localhost:3000/users
- Abrir con Live Server `index.html`
